# Sri Murugan Mess 🍛

A single-page restaurant website for Sri Murugan Mess, an authentic South Indian vegetarian restaurant in T. Nagar, Chennai (est. 1988). Customers can browse the menu, order food online, book a table, and pay using an in-app digital wallet — complete with login/signup and order/booking history.


# 🔗 Live Demo
https://gayathrit-06.github.io/FUTURE_FS_03/


# ✨ Features

🍽️ Browse menu by category (Breakfast, Meals, Tiffin, Chettinad, Sweets, Drinks).

🛒 Add items to cart and place orders online.

📋 Book a table with date, time slot, guest count & seating preference.

🔐 Login / Sign-up system with real password validation.

💰 In-app wallet — top up balance, pay for orders and table reservations.

📜 View past Order History and Booking History.

🗺️ Google Maps integration showing the restaurant's exact location.

🌙 Dark-themed, fully responsive design.

# 💳 How the Wallet Works

New accounts start with a free ₹500 wallet balance.

Ordering food deducts the cart total from the wallet.

Booking a table deducts a flat ₹50 reservation fee.

If the balance is too low, you'll be prompted to add money before the action completes.
